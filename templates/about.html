About Me

I'm a full-stack Javascript Developer

fundamentals, libraries, projects, client facing experience, project/code management, service design.

I learned at Founders and Coders

I was part of FAC's Winter 2019 cohort (FAC18). As well as being a supportive environment and network to learn web development, FAC promoted the value of open-source, tech for good and agile development.

I've spent the pandemic working on 'tech for good' projects.

Spurred on by enthusiasm for FAC's 'Tech for Better' programme, on graduation I teamed up with fellow FAC18 cohort member Bethany Scott to work on further Tech for Better projects as part fo a Founders and Coders team. This saw us work with a number of non-profit organisations and charities on projects ranging from web development to digital service design and product management learning programmes. These were often in partnership with CAST and the Catalyst network.

I've been learning Go in my spare time

I've wanted to add a new language beyond Javascript to my skillset for a while. I chose Go for a number of reasons, and have enjoyed getting to grips with new syntax and tooling. This site is my first major Go project (you can see the repo here). It uses Go's http and template packages from the standard library.
